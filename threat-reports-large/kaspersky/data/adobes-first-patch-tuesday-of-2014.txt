This month’s Adobe Patch Tuesday release sees fixes for Flash Player, Acrobat and Reader.
All vulnerabilities get the highest priority rating.
This means future exploits are likely.
The Flash Player bulletin was only announced today.
CVE-2014-0491 and CVE-2014-0492 both concern remote code execution vulnerabilities.
CVE-2014-0493, CVE-2014-0495 and CVE-2014-0496 affect Acrobat and Reader.
These CVEs also concern remote code execution vulnerabilities.
All of this month’s vulnerabilities were reported to Adobe directly.
Given the severity of the vulnerabilities we recommend applying these patches as soon as possible.
